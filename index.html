02 / HOW I WORK

Approach

The thesis behind everything above:

Grounded, or it doesn't ship

Every generated claim traces to a source chunk. When evidence is weak, the system says so — refusal behavior is a feature, tested like one.

Evaluated, not vibe-checked

If answer quality isn't measured, it's marketing. I build the eval harness before I trust the output — golden sets, regression scoring, adversarial cases.

Honest about failure modes

Finance punishes overconfidence. I document where tools break, what they must not do, and the biases that make backtests and benchmarks lie.
